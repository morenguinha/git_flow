# git_flow
### Principais comandos git:
---
1. `git init`
2. `git config`
3. `git clone https/link.of@remoterepository`
4. `git remote add origin https/link.of@remoterepository`
5. `git remote -v`
6. `git status`
1. `git log`
7. `git add .`
8. `git commit -m`
9. `git branch`
10. `git checkout`
11. `git flow init`
12. `git fetch`
13. `git pull`
14. `git merge`
15. `git push`
16. `git flow 'typebranch' publish 'tagbranch'`
